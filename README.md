# Практическая работа №7: CSS-фреймворк Bulma

**Тема:** Лендинг магазина кофе Revo (Редизайн).
**Фреймворк:** [Bulma](https://bulma.io/) (версия 0.9.4).

### Почему выбран Bulma:
1. Легкий и понятный синтаксис классов.
2. Работает только на CSS (не требует JS).
3. Отличная адаптивность "из коробки".
