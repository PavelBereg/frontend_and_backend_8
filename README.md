# Практическая работа №7: CSS-фреймворк Bulma

**Тема:** Лендинг онлайн-школы программирования "CodeStart".
**Фреймворк:** [Bulma](https://bulma.io/) (v0.9.4).

### Почему выбран Bulma:
1. **Flexbox-grid:** Очень удобно создавать сетки карточек курсов (`columns`).
2. **Компоненты:** Готовые красивые кнопки (`button is-link`), карточки (`card`), теги (`tag`).
3. **Чистота:** Не требует JavaScript, сайт грузится мгновенно.
